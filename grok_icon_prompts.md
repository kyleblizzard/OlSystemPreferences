## Style Notes for ALL prompts

Every icon should look like a **Mac OS X Snow Leopard (10.6) System Preferences icon** — photorealistic, skeuomorphic, glossy, with subtle reflections and soft drop shadows. These are small desktop icons, so the object should fill most of the frame with clean edges. Think 2009 Apple design language: polished materials, slight 3D depth, no flat design.
